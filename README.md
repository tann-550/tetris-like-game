## Tetris-like game

# Game Rules
1. Objective
The primary objective of the game is to score as much points as possible by clearing horizontal lines of blocks.
2. Mechanics
The game takes place on a 2D board, consisting of 10 columns and 20 rows.
There are 7 distinct Tetromino / shapes that fall from the top of the screen: 
- I-Piece: A straight line of 4 blocks.
- O-Piece: A 2x2 square block.
- T-Piece: A T-shaped block.
- J-Piece & L-Piece: L-shapes mirroring each other.
- S-Piece & Z-Piece: Zig-zag shapes mirroring each other.
3. Controls
Move Left: Left Arrow
Move Right: Right Arrow
Rotate: Up Arrow
Accelerates Drop: Down Arrow
Instantly Drop: Spacebar
Pause / UnPause: P
4. Scoring System
Points are awarded based on the number of lines cleared simultaneously:
- 1 Line = 100 points
5. Game Over Condition
The game ends when the stack of locked blocks reaches the top of the board, preventing the next spawned of Tetromino from entering the grid
